# Gisys Frontend Assignment

A small Angular application that fetches and displays employees from a public API.

## Tech Stack

- Angular 20 (Standalone API)
- Angular Signals for state management
- Angular Router (lazy-loaded routes)
- Tailwind CSS for styling
- ESLint + Prettier for code quality

## Features

- Employee list view
- Employee detail view
- Loading and error state handling
- Typed API layer with DTO → domain mapping
- Lazy-loaded routes
- Signal-based state management

## Getting Started

Install dependencies:

```bash
npm install
ng serve
